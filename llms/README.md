# Zoloti Maky - Landing Page

## How to View the Site

### Option 1: Double-click the HTML file
1. Find `index.html` in this folder
2. Double-click it to open in your web browser
3. That's it! You can now browse the site

### Option 2: Use a local server (better for development)
If you have issues with Option 1, you can:
1. Open Terminal
2. Navigate to this folder: `cd /Users/olena/Desktop/site`
3. Run: `python3 -m http.server 8000`
4. Open your browser and go to: `http://localhost:8000`
